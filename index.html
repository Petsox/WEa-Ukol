1. Co je služba WWW

Služba World Wide Web (zkráceně WWW) je systém propojených hypertextových dokumentů dostupných prostřednictvím internetu. Webové stránky se zobrazují pomocí webových prohlížečů (např. Google Chrome, Mozilla Firefox, Safari).

WWW vznikla v roce 1989 v Evropské organizaci pro jaderný výzkum (CERN). Jejím autorem je Sir Tim Berners-Lee.

Základní principy fungování WWW

  • Protokol HTTP/HTTPS: zajišťuje komunikaci mezi klientem (prohlížečem) a serverem.
  • URL adresa: jednoznačně identifikuje webovou stránku (např. https://www.example.com).
  • HTML: jazyk pro strukturování obsahu webových stránek.

2. Základní struktura HTML dokumentu

Každá HTML stránka má svou základní kostru. Příklad:

<!DOCTYPE html>
<html lang="cs">
  <head>
    <meta charset="UTF-8">
    <title>Název stránky</title>
  </head>
  <body>
    Obsah stránky
  </body>
</html>

Vysvětlení jednotlivých částí:

  • <!DOCTYPE html> – deklarace typu dokumentu (HTML5)
  • <html> – obaluje celý dokument
  • <head> – obsahuje metadata (např. kódování, název stránky)
  • <body> – viditelná část stránky

3. Unicode a HTML entity

Unicode

Unicode je standard, který umožňuje zapisovat znaky z různých jazyků a sad symbolů. Díky němu lze v HTML zobrazit znaky jako á, č, ř, ž nebo jiné speciální symboly.

Použitím <meta charset="UTF-8"> v hlavičce dokumentu zajistíme podporu Unicode znaků.

HTML entity

HTML entity se používají pro zobrazení speciálních znaků, které by jinak byly vyhodnoceny jako součást kódu. Například:

  • &lt; zobrazí znak: <
  • &gt; zobrazí znak: >
  • &amp; zobrazí znak: &
  • &nbsp; – nezalomitelná mezera
  • &copy; zobrazí: ©
  • &hearts; zobrazí: ♥

4. Blokové a inline elementy

Blokové elementy

Blokové prvky tvoří nový řádek a obvykle obsahují větší části obsahu.

  • <div> – univerzální kontejner
  • <p> – odstavec
  • <h1> až <h6> – nadpisy
  • <ul>, <ol>, <li> – seznamy
  • <section>, <article>, <footer>, <header> – sémantické části stránky

Inline elementy

Inline prvky se zobrazují v rámci řádku a nezpůsobují jeho zalomení.

  • <span> – univerzální inline kontejner
  • <strong> – zvýraznění textu (tučně)
  • <em> – zvýraznění textu (kurzíva)
  • <a> – odkaz
  • <code> – označení části kódu

7. Otázky a odpovědi

Co znamená zkratka HTML?
HTML = HyperText Markup Language, jazyk určený pro strukturování webových stránek.
Jaký je rozdíl mezi blokovým a inline elementem?
Blokové elementy zabírají celou šířku řádku, zatímco inline elementy pouze tolik místa, kolik potřebují.
Co dělá značka <meta charset="UTF-8">?
Zajišťuje správné zobrazení národních znaků pomocí Unicode (např. ě, š, č apod.).
K čemu slouží element <a>?
Vytváří hypertextový odkaz na jinou stránku nebo část stejného dokumentu.
