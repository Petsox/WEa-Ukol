1. Základní struktura HTML dokumentu

Každá webová stránka začíná deklarací typu a obsahuje dvě hlavní části: <head> a <body>.

<!DOCTYPE html>
<html lang="cs">
  <head>
    <meta charset="UTF-8">
    <title>Název stránky</title>
  </head>
  <body>
    Obsah stránky
  </body>
</html>
Tip: Vždy používejte <meta charset="UTF-8"> pro správné zobrazení znaků!

2. Principy fungování WWW

  • HTTP/HTTPS: Protokoly pro přenos dat mezi klientem a serverem.
  • URL: Jedinečná adresa každé stránky (např. https://www.wikipedia.org).
  • Webový prohlížeč: Program pro zobrazení webových stránek (Chrome, Firefox, Edge).
  • Server: Počítač, který poskytuje obsah webu.

Služba WWW byla vytvořena v CERNu roku 1989 Timem Berners-Leem.

3. Elementy HTML

Blokové elementy

  • <div> – univerzální kontejner
  • <p> – odstavec
  • <h1> až <h6> – nadpisy
  • <section>, <article>, <footer>, <header>

Inline elementy

  • <span> – univerzální inline kontejner
  • <strong> – tučný text
  • <em> – kurzíva
  • <a> – odkaz
  • <code> – kód

6. Unicode a HTML entity

Unicode

Standard pro zápis znaků z různých jazyků. V HTML používejte <meta charset="UTF-8">.

HTML entity

  • &lt; zobrazí <
  • &gt; zobrazí >
  • &amp; zobrazí &
  • &copy; zobrazí ©
  • &hearts; zobrazí ♥

7. Nejčastější otázky (FAQ)

Co znamená HTML?
HyperText Markup Language – jazyk pro tvorbu webových stránek.
Jaký je rozdíl mezi blokovým a inline elementem?
Blokové elementy tvoří nový řádek, inline elementy se zobrazují v rámci řádku.
K čemu slouží <meta charset="UTF-8">?
Zajišťuje správné zobrazení znaků pomocí Unicode.
Jak vytvořím odkaz?
Použijte <a href="adresa">Text odkazu</a>.
Co je CSS?
Cascading Style Sheets – jazyk pro stylování webových stránek.
Jak přidám obrázek?
Použijte <img src="adresa.jpg" alt="Popis">.
